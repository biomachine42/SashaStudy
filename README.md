# 🔮 Предсказание на день

Простое и красивое веб-приложение, которое выдаёт шуточное предсказание по нажатию кнопки. Из бытовых и узнаваемых сюжетов: выпить кофе с подругой, найти деньги в кармане куртки, не потерять сумку в метро, не поругаться с бабушкой на остановке.

Всего **100 уникальных предсказаний** — каждый клик новый.

## ✨ Что умеет

- Генерирует случайное предсказание по нажатию кнопки
- Показывает «число дня», «цвет дня» и «настрой» 🍀
- Анимированный звёздный фон
- Работает на телефоне и компьютере
- Не требует интернета после открытия — всё в одном файле

## 🚀 Как запустить

Просто открой файл `index.html` двойным кликом — приложение откроется в браузере. Больше ничего устанавливать не нужно.

## 🌐 Как выложить в интернет

### Вариант 1. Netlify Drop (самый быстрый)

1. Открой [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащи файл `index.html` в окно браузера
3. Получи готовую ссылку на сайт

### Вариант 2. GitHub Pages

1. Создай публичный репозиторий на GitHub
2. Загрузи `index.html` (Add file → Upload files → Commit)
3. В **Settings → Pages** выбери ветку `main` и папку `/ (root)`, нажми **Save**
4. Через пару минут сайт будет доступен по адресу `https://твой-логин.github.io/название-репозитория/`

## 🛠 Как устроено

Всё приложение умещается в одном файле `index.html` и состоит из трёх частей:

- **HTML** — структура страницы (заголовок, кнопка, блоки с результатом)
- **CSS** — оформление (цвета, скруглённые карточки, звёздный фон). Основной цвет — `#2A9FFF`
- **JavaScript** — логика: при клике случайно выбирает предсказание, число, цвет и эмодзи настроения

Чтобы добавить своё предсказание, открой `index.html` в редакторе, найди список `PREDICTIONS` и допиши строку.

## 📁 Файлы

- `index.html` — версия для публикации (GitHub Pages / Netlify)
- `predskazanie.html` — та же версия с понятным именем для локального открытия
