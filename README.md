# 🪩 Предсказание на день

Небольшой сайт с двумя гаданиями: шуточное предсказание на день и карта дня по Таро Уэйта. Тёмная «космическая» тема, анимированный звёздный фон, общий подвал на всех страницах.

## ✨ Страницы

- **Предсказание** (`index.html`) — по нажатию кнопки выдаёт случайное бытовое предсказание из **100 вариантов** (кофе с подругой, найденные деньги, бабушка на остановке и т.д.), а также «число дня», «цвет дня» и «настрой».
- **Карта дня** (`karta-dnya.html`) — вытягивает карту из **полной колоды Таро Уэйта (78 карт)**: 22 старших аркана и 56 младших (Жезлы, Кубки, Мечи, Пентакли). При вытягивании — «магия»: карта дрожит, светится и осыпается искрами, затем переворачивается с толкованием на день.

Между страницами можно переключаться через меню вверху.

## 🚀 Как запустить

Открой `index.html` двойным кликом — сайт откроется в браузере. Устанавливать ничего не нужно.

## 🌐 Как выложить в интернет

Важно: загружать нужно **все файлы сразу** (`index.html`, `karta-dnya.html`, `style.css`, `app.js`) и держать их рядом — иначе стили и звёздный фон не подхватятся.

### Netlify Drop (самый быстрый)

1. Открой [app.netlify.com/drop](https://app.netlify.com/drop)
2. Перетащи туда всю папку с файлами целиком
3. Получи готовую ссылку на сайт

### GitHub Pages

1. Создай публичный репозиторий на GitHub
2. Загрузи все файлы проекта (Add file → Upload files → Commit) — они должны лежать рядом
3. В **Settings → Pages** выбери ветку `main` и папку `/ (root)`, нажми **Save**
4. Через пару минут сайт будет доступен по адресу `https://твой-логин.github.io/название-репозитория/`

## 🛠 Как устроено

Сайт построен на общих файлах, чтобы не дублировать код:

- `style.css` — все общие стили (тема, карточка, меню, подвал, цвет `#2A9FFF`). Меняешь в одном месте — применяется на всех страницах.
- `app.js` — анимированный звёздный фон, общий для всех страниц.
- HTML-файлы подключают эти два файла и содержат только то, что уникально для конкретной страницы (тексты, разметку и свою логику).

Иконка вкладки (favicon) встроена прямо в HTML, поэтому потеряться не может.

Чтобы добавить своё предсказание, открой `index.html` и допиши строку в список `PREDICTIONS`. Колода Таро в `karta-dnya.html` собирается из массивов `MAJORS`, `RANKS` и `SUITS`.

### Общий подвал

Внизу каждой страницы — подвал с надписью **let's rock** и ссылкой на Telegram. Стили подвала лежат в `style.css`, а сама разметка помечена комментарием `<!-- ОБЩИЙ ПОДВАЛ -->` и копируется при добавлении новой страницы.

## ♿ Доступность и качество

- Поддержка системной настройки «уменьшить движение» (`prefers-reduced-motion`) — анимации отключаются
- Блоки результата помечены `aria-live`; название выпавшей карты Таро озвучивается скринридерам
- Декоративный звёздный фон скрыт от скринридеров
- Одинаковая высота карточки на всех страницах — при переключении вкладок ничего не «прыгает»
- Адаптивная вёрстка — корректно работает на телефоне и компьютере
